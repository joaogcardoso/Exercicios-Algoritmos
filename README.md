# Exercicios-Algoritmos
1) Faça um algoritmo que leia o raio de um círculo, calcule a sua área e informe o resultado com uma
mensagem.
2) Faça um algoritmo que leia um número, informe o número lido e, em seguida, informe se o número é
positivo, negativo ou zero.

4) Faça um algoritmo que leia um número, informe o número lido e, em seguida, informe se o número
lido é par ou ímpar.

5) Faça um algoritmo que leia um número, informe o número lido e, em seguida, mostre o resultado de
sua tabuada de 1 a 10. (dica: use estrutura de repetição).

6) Faça um algoritmo que leia um número, informe o número lido e, em seguida, imprima uma listagem
do número informado até 100.

7) Faça um algoritmo para ler 10 números, e em seguida, informar o resultado da soma dos 10 números
informados.

8) Faça um algoritmo para ler 3 números, e em seguida, informar o maior entre os 3 números
informados.

9) Faça um algoritmo para ler 3 notas, calcular e exibir a média. Em seguida, o programa deverá emitir
uma mensagem informado se o aluno está Aprovado, Reprovado ou de Recuperação. Considere que:
Se ≥ 7: aprovado. 5 a 6.9: recuperação. <5: reprovado.

10) Faça um algoritmo para calcular o fatorial de um número.

11) Faça um algoritmo para exibir os números pares até 100.

12) Faça um algoritmo que deverá ter um número secreto. O usuário deverá tentar adivinhar o número
secreto, informando um número. O algoritmo sempre deverá solicitar que o número seja informado e
terminará sua execução quando o usuário acertar.

13) Faça um algoritmo que deverá realizar soma de números até que um número negativo seja
informado.
